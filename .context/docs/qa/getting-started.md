# How do I set up and run this project?

## Getting Started

### Prerequisites

- Node.js (LTS version recommended)
- npm

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd aurelia

# Install dependencies
npm install
```

### Running

```bash
# See package.json for available scripts
npm run <script-name>
```
