# Project Overview: Remote-control (Elite Template)

Este projeto é um workspace canônico projetado para orquestração de múltiplos agentes de IA (Antigravity, Claude Code, Codex). Ele foca em segurança, autonomia e densidade de contexto.

## Objetivos Principais
- Fornecer um boilerplate robusto para desenvolvimento AI-First.
- Implementar Tiers de Autonomia (A, B, C) para controle de riscos.
- Centralizar a governança em `.agents/` e a memória em `.context/`.

## Stack de Contexto
- **Orquestrador**: Antigravity IDE.
- **Motores**: Claude 3.5 Sonnet / Codex.
- **MCP**: ai-context (antigravity-mcp).
- **Injeção**: \`ag-init\` (bootstrap universal).
