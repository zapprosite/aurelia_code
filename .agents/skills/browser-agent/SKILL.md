# 🌐 Browser Agent: Sovereign Web Control

Habilita interações via Playwright para navegação, extração de dados e automação de interfaces web.

## 📍 Quando usar
- Scraping de documentação externa.
- Testes E2E de interfaces locais (ULTRATRINK).
- Automação de logins ou downloads de mídia.
