# Decision Matrix

## Use Antigravity Chat

- editar um unico arquivo de config
- explicar uma opcao de provider
- localizar uma flag
- resumir log ou erro
- montar `curl`, `jq`, `rg`
- propor checklist rapido

## Use Aurelia CLI

- mudar codigo de runtime
- tocar mais de 2 arquivos acoplados
- validar comportamento real
- rodar testes
- commitar
- atualizar `.context/`

## Escalar para humano

- segredo novo
- operacao destrutiva
- rede ou firewall
- deploy
- mudanca sem rollback simples
