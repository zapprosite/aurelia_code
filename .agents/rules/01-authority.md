# 📜 Regra 01: Autoridade Única

Este repositório opera sob um modelo de autoridade centralizada.

<directives>
1. **Fonte de Verdade**: O arquivo `AGENTS.md` na raiz é a constituição soberana. Nenhuma regra local pode contradizê-lo.
2. **Orquestração**: O Antigravity IDE é o supervisor global. Motores como Claude e Codex atuam como braços executores sob supervisão.
3. **Escopo de Decisão**: Mudanças estruturais exigem consulta ao `PRD.md` e registro via ADR (Architectural Decision Record).
</directives>
