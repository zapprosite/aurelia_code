# 🛰️ Guia Rápido: Antigravity Kit v2.0 (Sovereign 2026)

Este guia orienta o uso da nova frota de agentes e comandos industriais.

## 1. Comandos de Fluxo (Power Commands)
Sempre inicie uma tarefa utilizando um destes comandos:

- **/pm**: Define os requisitos (PRD).
- **/architect**: Cria a Tech Spec (Design).
- **/dev**: Inicia a implementação técnica.
- **/scaffold**: Gera módulos completos (Zod + ORM).
- **/qa**: Validação e auditoria de qualidade.

## 2. Orquestração (/master-skill)
- `/master-skill search {termo}`: Procura habilidades.
- `/master-skill import {skill}`: Instala novas skills.

## 3. Entrega (Sovereign Git)
- **/git-turbo**: Commit, push, merge e tag automáticos.
- **/git-ship**: Staging e Pull Request profissional.

## 🛡️ Regra de Ouro
Use sempre `{chave-para-env}` para segredos. O AG-Kit v2.0 protege sua soberania.
