# ADR 20260328: Jarvis Tutor - Escuta Tudo 24/7

## Status
🔴 Implementado

## Contexto
O Will precisa de um **Jarvis Tutor** que:
- **Escuta tudo** - sem wake word, sempre ouvindo
- **Vive 24/7** - no backoffice, nunca desliga
- **É tutor** - ajuda, orienta, ensina
- **Responde em português brasileiro**

## Decisões Arquiteturais

### 1. Sempre Ligado (No Wake Word)
```go
// Voice loop SEM wake word - escuta tudo
// Config.WakePhrase = "" (vazio)
```

### 2. BK (Backoffice) Service
```bash
# Install
sudo cp docs/systemd/jarvis-tutor.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable --now jarvis-tutor

# Params (substituir USER_ID e CHAT_ID)
ExecStart=/home/will/aurelia/aurelia tutor --user=USER_ID --chat=CHAT_ID
```

### 3. Tutor Mode
```go
const TutorSystemPrompt = `Você é o Jarvis, um tutor paciente e prestativo.
- Explica passo a passo
- Pergunta para confirmar entendimento
- Dá exemplos práticos
- Corrige com gentileza
- Responde em português brasileiro
`
```

## Comandos

```bash
# Rodar manualmente
./aurelia tutor --user=123456 --chat=789012

# Instalar como serviço (BK 24/7)
sudo cp docs/systemd/jarvis-tutor.service /etc/systemd/system/
sudo systemctl edit jarvis-tutor  # EDITAR USER_ID e CHAT_ID
sudo systemctl enable --now jarvis-tutor

# Status
systemctl status jarvis-tutor

# Logs
journalctl -u jarvis-tutor -f
```

## Consequências

### Positivas
- Sempre disponível para responder
- Mais natural (sem wake word)
- Contexto contínuo da conversa
- Tutor paciente e explicativo

### Negativas
- Pode responder a ruídos de fundo
- Mais uso de recursos (STT sempre ativo)
- Necessita de boas políticas de silêncio
