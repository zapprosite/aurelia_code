# Comandos Master Skill

Use o comando `/master skill` para gerenciar o ambiente de desenvolvimento de forma sênior e otimizada.

## Invocação

- `/master skill init`: Setup inicial do agente e do diretório de skills globais.
- `/master skill install [framework]`: Instalação automatizada de Antigravity Kit, BMED ou Specit.
- `/master skill search [query]`: Descoberta de ferramentas no "Skill Store" local.
- `/master skill import [skill-name]`: Importação seletiva de lógica para o projeto atual.

## Governança

Este comando opera via skill global localizada em `~/.gemini/antigravity/skills/master-skill/`. Respeita o contrato de camadas e silêncio industrial (Zero Hardcode).
