# Master Skill: Sovereign Bootstrap 🛠️🌍

Esta skill governa a industrialização de novos projetos no ecossistema Aurélia, garantindo paridade de governança em segundos.

## Comandos
- `/master-skill init`: Inicializa o repositório atual com `.context/`, `.agent/` e ADRs base.
- `/master-skill install {kit}`: Instala `antigravity`, `bmad` ou `spec-kit`.
- `/master-skill sync`: Sincroniza skills do catálogo central.

## Lógica de Execução
1.  **Auditoria**: Verifica se já existe uma governança ativa para evitar conflitos.
2.  **Scaffolding**: Cria a árvore de diretórios oficial.
3.  **Kits**: Copia ou linka os arquivos de regras e comportamentos dos agentes de elite.

## Configuração Global
Os kits de IA residem em `~/.aurelia/kits/` e são injetados em cada novo projeto para manter a soberania e performance SOTA 2026.
