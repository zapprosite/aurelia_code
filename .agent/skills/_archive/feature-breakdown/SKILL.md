# 📉 Feature Breakdown: Sovereign Planning 2026

Habilita o Antigravity a transformar ideias vagas em planos de ação industriais (`task.md`).

1. **Objetivo**: Defina o que é "sucesso".
2. **Tasks**: Quebre em tarefas de no máximo 30 min cada.
3. **Dependencies**: Identifique o que bloqueia o que.

## 📍 Quando usar
- No início de cada novo ciclo de planejamento (`P`).
- Para estimar o tempo necessário para um novo slice.
