# Communication Refinery 💎

Skill especializada em refinamento iterativo das saídas da Aurélia (Markdown + Voz).

## Propósito
Esta skill permite que desenvolvedores e operadores testem a estética das respostas em Markdown e a qualidade da síntese de voz Kokoro (PT-BR) em um loop fechado no Telegram Desktop.

## Ferramentas
- `./refine.sh "<texto>"`: Gera áudio e envia para o Telegram.

## Configuração
- **Voz Padrão**: `af_sarah` (PT-BR aliased)
- **Formato**: Opus (Ogg)
- **Destino**: ChatID `7220607041` (Will)

## Como Refinar
1. Execute o script com uma proposta de resposta.
2. Ouça a entonação no Telegram.
3. Ajuste pontuação ou tags Markdown conforme necessário.
4. Repita até atingir o padrão "Soberano 2026".
