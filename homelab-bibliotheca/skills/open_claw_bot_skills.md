# Open Claw Bot Skills Repository

Este documento resume as informações coletadas do repositório de skills do Open Claw Bot.

## Localização do Repositório

*   [https://github.com/openclaw/skills](https://github.com/openclaw/skills)

## Estrutura Original

*   `core/`: Skills essenciais usadas por todos os bots.
*   `skills/`: Skills específicas para bots particulares.
*   `tools/`: Ferramentas utilizadas pelas skills.

## Importação Local

As skills foram baixadas e estão sendo mapeadas para integração com a Aurélia em `.agent/skills/` e `homelab-bibliotheca/skills/`.
